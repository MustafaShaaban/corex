## What is Corex?

Corex is a WordPress framework with a clean, layered architecture: a self-booting core
(`corex-core`), a PSR-11 container, thin controllers over fat services over repositories,
dynamic FSE blocks as reusable components, a shared-schema forms engine, a fluent
QueryBuilder, a `wp corex make:*` CLI, and runtime design tokens (`theme.json` → CSS
variables → per-site `brand.json`).

Target: **WordPress 7.0+, PHP 8.3+**, FSE block themes. Namespace `Corex\`, CLI
`wp corex`, CSS prefix `--corex-`.
