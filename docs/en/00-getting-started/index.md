# Getting started

Local setup for a developer with **zero** prior Corex experience. Pick your operating system — each guide
introduces every tool it needs and ends with a booting site.

> **Status:** this section is scaffolded; the per-OS guides are authored in **phase D2** (see
> [`specs/028-developer-handbook/tasks.md`](../../../specs/028-developer-handbook/tasks.md)).

## Guides (planned)

| Guide | For |
|---|---|
| `windows-wamp.md` | Windows + WAMP |
| `windows-xampp.md` | Windows + XAMPP |
| `linux.md` | Linux (Ubuntu / Debian) |
| `macos.md` | macOS |
| `wp-env.md` | wp-env (Docker) — the zero-install option |

Each follows the getting-started shape in
[`contracts/page-contract.md`](../../../specs/028-developer-handbook/contracts/page-contract.md): tool intros,
command → expected output, the monorepo → `wp-content/` mapping, and a boot verification.
