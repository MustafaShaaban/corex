## Summary

A static documentation site (Astro + Starlight) under `docs-app/`: a configured site shell (sidebar,
search, theme) plus hand-written Markdown/MDX pages describing the real code, consuming the generated
per-class reference pages (spec 019). Builds to `dist/` with `npm run build`; runs locally with `npm run dev`.

## Technical Context

**Language/Version**: JS/TS (Astro 4.x), Markdown/MDX content. **Primary Dependencies**: `astro`,
`@astrojs/starlight` (bundles Pagefind search). **Testing**: the production build (`npm run build`) is the
gate — green build + expected page set + search index. **Project Type**: static site (separate npm project,
not a workspace member). **Constraints**: docs describe real code (docs-guard); RTL-ready; `node_modules`/
`dist`/generated `reference/*` git-ignored.
